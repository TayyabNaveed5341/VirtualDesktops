Program Rules

Programs can be set up with specific rules to further customize your virtual desktop experience. Create a new rule by entering in the name of the executable with or without the file extension (.exe). For example, "taskmgr" is preset to stay "sticky" on all desktops because "taskmgr" is the name of Task Manager's executable file. You can find out a program's executable name quickly by opening task manager (Control-Shift-Escape), right-clicking on the application you want and selecting "Go to Process". The name listed in the first column is the executable file's name.

Sticky Programs

Sticky programs will always appear on all desktops. In effect, making a program sticky causes Finestra to completely ignore its existence.

Stay on Desktop X

By making a program stay on a single desktop, Finestra will watch for new windows to open and automatically move them to the proper desktop.

Minimize before Hiding

Some programs don't like to be hidden by Finestra on virtual desktop switches. Therefore, they can be made to minimize immediately before hiding. They will then be automatically restored when they are unhidden. Notable programs that must use this include the Adobe Illustrator CS3 and Adobe Photoshop CS3. Rules have already been added for these programs.
