Shortcuts

When Finestra is started, it automatically assigns itself some shortcut keys, or "hotkeys". The defaults are:

  • Show the Switcher Screen - Windows Key + Z
  • Switch to Desktop 1...9 - Windows Key + Numpad 1...9 (with numlock)
  • Window Menus - Windows Key + W
  • Switch to the Next Desktop - Windows Key + Right/Down Arrow
  • Switch to the Previous Desktop - Windows Key + Left/Up Arrow
  • Send Window to Desktop 1...9 - Windows Key + Control + Numpad 1...9 (with numlock)
  • Show all Open Windows - Windows Key + F9
  • Show the Mini-Preview at the Cursor - Windows Key + P

You can also easily customize these keyboard shortcuts by right-clicking on the system tray icon and selecting "Options". There are two tabs labeled "Hotkeys" and "More Hotkeys" that contain these options.
