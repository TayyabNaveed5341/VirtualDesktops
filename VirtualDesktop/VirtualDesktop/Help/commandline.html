Command Line Options

Vista/XP Virtual Desktops accepts a number of command line arguments in order to allow its use in scripts and shortcuts. It is recommended to start Finestra before using command line arguments (or to just let it start automatically without command line arguments).

/switcher

Opens the switcher.

/desk:<desktop name or number>

Switches to the given desktop. Note that desktop numbers start at 1 and that if the desktop name has a space in it, the entire option must be enclosed in quotations. For example:

  • "Virtual Desktops.exe" "/desk:A Long Desktop Name with Spaces"
  • "Virtual Desktops.exe" /desk:2

<desktop name or number>:<program or process ID>

Opens a program onto the specified desktop or forces a currently open program onto the specified desktop. Once again, if the program name or the desktop name has spaces, the entire option must be enclosed in quotes. To stop forcing a program to a desktop, simply use 0 as the desktop number. For example:

  • "Virtual Desktops.exe" 2:notepad.exe
  • "Virtual Desktops.exe" Work:word.exe
  • "Virtual Desktops.exe" "Development:my program.exe"
