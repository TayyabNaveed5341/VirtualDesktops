Window Menus

Vista/XP Virtual Desktops contains a feature called "Window Menus". By default, you call up a window menu by activating a window and pressing Windows Key + W. A menu should pop up under your mouse cursor. These options affect just the activated window. The following commands are contained in the window menu.

  • Show Thumbnail - Opens a window that displays a live thumbnail of the selected window in Windows Vista.
  • Make Window Sticky/Unstick Window - Making a window sticky causes it to appear on all desktops.
  • Send to Desktop X - Immediately sends the window to desktop X. There are also hotkeys for this. By default they are Windows Key + Control + Numpad 1...9.
